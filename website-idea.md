### WEBSITE — nyyraa.com

**Platform recommendation: Shopify (Rs.1,994/month) or WooCommerce (cheaper)**

**Shopify Setup Steps:**
1. Go to shopify.com → Start free trial (3 months Rs.20)
2. Store name: Nyyraa Beauty
3. Connect domain: nyyraa.com → Settings → Domains → Connect existing
4. Theme: Dawn (free, minimal, perfect for beauty brand)
5. Colors: Update theme colors to Nyyraa palette

**Website Pages to Create:**
```
HOME          → Hero banner + bestsellers + brand story
SHOP          → All products with filters
ABOUT         → Korean beauty + Indian skin story
INGREDIENTS   → Science-backed ingredients page (builds trust)
REVIEWS       → Customer testimonials
CONTACT       → info@nyyraa.com + Instagram
```

**Home Page Structure:**
```
1. Hero Banner     : "Glow Like Never Before" + Shop Now CTA
2. Trust Bar       : Korean Inspired | Paraben Free | Cruelty Free | Derm Tested
3. Bestsellers     : 3-4 products with Add to Cart
4. Brand Story     : "For skin like yours" — Indian skin focus
5. Ingredients     : Hydrolyzed Collagen | Hyaluronic Acid | Niacinamide
6. Reviews         : 5-star reviews section
7. Instagram Feed  : @nyyraabeauty posts
8. Footer          : Links + email signup
```

**Essential Shopify Apps to Install (all free tier):**
- **Vitals** — Reviews + upsells + trust badges
- **Klaviyo** — Email marketing automation
- **Judge.me** — Product reviews
- **Loox** — Photo reviews
- **Tidio** — Live chat

**SEO Setup for nyyraa.com:**
- Title: "Nyyraa Beauty | Korean Skincare for Indian Skin | Collagen Masks, Sunscreen"
- Meta description: "Premium Korean-inspired skincare designed for Indian skin. Paraben-free, cruelty-free. Shop Collagen Face Masks, SPF 50 Sunscreen, Lip Tints."
- Install: SEO Manager app on Shopify
